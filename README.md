# Siw Årolilja – Portefølje

Dette er min personlige portefølje, laget for å vise frem mine UX/UI- og frontend-prosjekter. Porteføljen kombinerer moderne webteknologi med interaktive funksjoner for å demonstrere mitt arbeid med design, struktur og utvikling.

---

##  Innhold

- **Hero-seksjon:** Presenterer meg selv og mitt fokusområde.
- **Prosjektseksjon:** Viser frem tidligere prosjekter, med "Vis mer"-modaler for detaljer.
- **Om meg:** Kort introduksjon om min bakgrunn og kompetanse.
- **Kontakt:** Mulighet til å sende e-post eller besøke LinkedIn-profil.

---

##  Teknologier

- HTML5
- CSS3 (med CSS variables og flex/grid)
- JavaScript (ES6+)
- Intersection Observer API for subtile animasjoner
- Dark/light mode toggle med localStorage

---

##  Funksjoner

- Responsive design som fungerer på mobil, nettbrett og desktop.
- Dark/light mode toggle.
- Fade-in animasjoner på kort når de kommer inn i viewport.
- Modaler for detaljerte prosjektbeskrivelser.
- Beskyttet e-postlink for å unngå spam-bots.
